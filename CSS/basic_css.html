CSS :-

  • CSS stands for Cascading Style Sheets
  • It gives style to raw HTML
  • It is used to give style to our webpages
  • It is used to make websites responsive
  • CSS is used to provide design to the structure provided by the HTML to a website

CSS Syntax :-

selector { property : value; }
For Ex: p { color : blue; }
here,
p is the paragraph selector (a selector can be for a class / id / element tag directly),
color is the property that tells the colour of the text,
blue is the value that tells us that the colour should be set to blue.

Three ways to add CSS to the markup (HTML Code) :-

  • Inline CSS :-
    • CSS is added to the element directly using style attribute.
  • Internal CSS :-
    • CSS is written inside the head element in <style> tags.
  • External CSS :-
    • CSS is kept seperately inside a .css style sheet
