CSS Fonts :-

There are generally two types of fonts.
1) Web safe fonts, they are the fonts already available on your operating system.
2) Web fonts, they have to imported for using.

These fonts can be used through css by defining font-family attribute.

There is a concept of font-stacking as we have used in our styling. We provide the font-family attribute with a list of fonts we prefer. In that case if first one is not available in the user's browser it searches for the next font and continues this way.
